# フォーマット文字列

```ruby
sprintf( "%02d", 5 )
=>"05"

"%02d" % 5
=>"05"
```

## フラグ
\#：２、８、１６進の指示子(b, B, o, x, X)と組み合わせ、0b, 0B, 0, 0x, 0Xのプレフィックスを付加
```ruby
"%#b" % 10
=>"0b1010"

"%#o" % 0
=>"0"
```
+：数値の指示子(d, i, b, B, o, x, X, u, f, e, E, g, G)の場合、符号を付加
```ruby
"%d" % 1
=>"1"
"%+d" % 1
=>"+1"
"%d" % -1
=>"-1"
"%+d" % -1
=>"-1"
```

半角空白：数値の指示子(d, i, b, B, o, x, X, u, f, e, E, g, G)の場合、'+'の場合１文字空白を付加
```ruby
"%+d" % 1
=>"+1"
"% d" % 1
=>" 1"
```

0：数値の指示子の一部(d, i, b, B, o, x, X, u, f, g, G)の場合、ゼロを付加
```ruby
"%02d" % 1
=>"01"
"%010x" % 10
=>"000000000a"
```

## 指示子
### 文字、文字列
c：引数の数値（０～２５５）を文字コードとみなし対応する文字を出力
```ruby
"%c" % 97
=>"a"
```

s：文字列を出力

p：Object#inspectの結果を出力

### 整数
d：引数の数値を１０進数表現で出力
```ruby
"%02d" % 7
=>"07"
```

b, B：整数を２進数表現で出力
```ruby
"%b" % 10
=>"1010"
```

o：整数を８進数表現で出力
```ruby
"%#o" % 10
=>"012"
```

x, X：整数を１６進数表現で出力
```ruby
"%#f" % 15
=>"0xf"
"%#x" % -1
=>"0x..f"
```

### 実数
f：小数点表現
```ruby
"%f" % 1
=>"1.000000"

"%f" % 1.0/0
=>"inf"
"%f" % -1.0/0
=>"-inf"
"%f" % 0.0/0
=>"nan"
```

e：指数表現
```ruby
"%e" % 1.2
=>"1.200000e+01"
"%e" % 0.5
=>"5.000000e-01"
```
